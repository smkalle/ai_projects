🗂️ Feature Tour

File & Batch Transcription

Upload wav/mp3/m4a files. Select language and punctuation. Get timestamped transcripts. Queue multiple files and download all results as a ZIP.

transcribing earnings_call_demo.wav87%

Real-Time Microphone

Live speech-to-text from your microphone. 16kHz mono, chunked every ~5s. Language selected per session. Rolling transcript display updates in under 3 seconds.

Recording ●  English — 00:23 elapsed

"The patient's blood pressure was"
"elevated at 145 over 92 and we"
"recommend starting lisinopril"

Transcript Library

All transcripts saved locally in SQLite. Search by keyword. Tag by project or patient cohort. Inline edit the transcript text. Export as .txt or .docx.

Data Analysis

Import clinical CSV/Excel datasets. Preview rows, filter by column values, compute descriptive statistics (n, mean, median, SD). Generate interactive Plotly charts.

WER Benchmarking

Upload your own audio + reference transcript to measure accuracy. See WER (word error rate) and RTFx (real-time factor) per language. Track history across multiple runs.

Sessions & Export

Named workspaces auto-save every 60 seconds. Resume exactly where you left off. Export the full session as a formatted PDF including transcript, metadata, and charts.

⚡ Quick Install

# Clone and enter the project
git clone <repo-url>
cd medical-workbench

# Install dependencies (uv handles everything)
./setup.sh

# Optional: set API key for cloud mode
$EDITOR .env

# Launch the workbench
./run.sh

Opens at http://localhost:8501 — no GPU required for API mode.
