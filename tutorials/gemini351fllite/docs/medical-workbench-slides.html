What is Agentic AI?

Instead of one AI answering one question, multiple specialized agents decompose a problem, coordinate their work, and refine results autonomously.

Google Agent Development Kit

Four Agent Primitives

Sequential
SequentialAgent
Run stages in strict order. Each completes before the next starts.
Parallel
ParallelAgent
Run independent agents concurrently. Isolated state per branch.
Loop
LoopAgent
Iterate until a condition is met. Generator + Critic pattern.
Router
LlmAgent (coordinator)
LLM dynamically selects which sub-agent to delegate to.

Stage 1

Triage

  • LlmAgent with LOW thinking for speed
  • Classifies urgency: Emergent, Urgent, Semi-Urgent, Non-Urgent
  • Calls calculate_bmi tool
  • Writes to triage_result output key
  • Hardcoded red flags fire before any AI call

Stage 2a

Symptom Analysis

  • Extracts structured symptom clusters
  • Interprets lab values
  • Calls calculate_egfr
  • Uses include_contents='none'

Stage 2b — runs concurrently

Imaging Analysis

  • Analyzes uploaded images via Gemini Vision
  • ABCDE criteria for skin lesions
  • Outputs ImagingObservation schema
  • Parallel isolation via session state

Stage 3

Diagnosis Refinement Loop

  • LoopAgent with max 3 iterations
  • Generator uses HIGH thinking for deep reasoning
  • Critic evaluates confidence + provides feedback
  • Exit via tool_context.actions.escalate = True
  • Exits when confidence ≥ 0.80

Stage 4

Specialist Routing

  • LLM-based routing — not rules
  • 4 specialists: Cardiology, Dermatology, Pediatrics, General Medicine
  • ADK's transfer_to_agent mechanism
  • Specialists access drug interaction + contraindication tools

Stage 5

Case Summary

  • Aggregates all prior stage outputs
  • Generates SOAP-format clinical note
  • Subjective, Objective, Assessment, Plan
  • Calls calculate_risk_score (Wells, TIMI, CURB-65)
  • Supports streaming output in the UI

def build_pipeline() -> SequentialAgent:
    """5-stage medical diagnostic pipeline."""
    return SequentialAgent(
        name="MedicalPipeline",
        sub_agents=[
            build_triage_agent(),        # LlmAgent
            build_concurrent_analysis(), # ParallelAgent
            build_diagnosis_loop(),      # LoopAgent
            build_specialist_router(),   # LlmAgent (coordinator)
            build_summary_agent(),       # LlmAgent
        ],
    )
