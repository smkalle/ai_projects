# Text-to-SQL Tutorial Project

A tutorials directory for building an agentic natural language-to-SQL system using LangChain Deep Agents.

## Contents

### Documentation

| File | Description |
|------|-------------|
| `tex2sql_tutorial.md` | Full hands-on tutorial for building a production-grade text-to-SQL agent |
| `wardops-brd-prd.md` | Business and product requirements for "WardOps Dashboard" |

### Applications

| Folder | Description |
|--------|-------------|
| `text2sql_app/` | Sample application demonstrating the text2sql concepts |

## Quick Start

```bash
# Install dependencies
pip install text2sql

# Follow the tutorial
cat tex2sql_tutorial.md
```
